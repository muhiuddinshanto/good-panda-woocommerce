# Good Panda E-commerce Landing Page

## Overview

Good Panda is a modern and responsive e-commerce landing page built using only HTML5 and CSS3. The website showcases featured products, categories, and a newsletter section with a clean and colorful user interface.

The design is inspired by a product-focused online store and includes sections for featured electronics, shopping categories, shoes, backpacks, and email subscription.

## Why These Technologies Were Used

### HTML5

Used to build the structure of the website, including:

* Navigation bar
* Hero section
* Category section
* Product cards
* Newsletter section

### CSS3

Used for all visual styling such as:

* Colors
* Typography
* Spacing
* Borders
* Shadows
* Responsive layout

### Flexbox

Used to align items and create flexible layouts in sections like:

* Navigation bar
* Hero section
* Categories
* Product cards
* Newsletter form

### Media Queries

Used to make the website responsive across different screen sizes:

* Laptop
* Tablet
* Mobile

### Google Fonts

Two fonts were used:

* Poppins
* Inter

These fonts help create a modern and clean appearance.

## Responsive Design

The website supports three different screen sizes:

| Device | Screen Width |
| ------ | ------------ |
| Laptop | Below 1200px |
| Tablet | Below 992px  |
| Mobile | Below 640px  |

### Responsive Changes

#### Laptop

* Hero image becomes smaller
* Categories wrap automatically
* Newsletter section padding becomes smaller

#### Tablet

* Hero section changes into a stacked layout
* Product cards wrap into multiple rows
* Newsletter section becomes narrower

#### Mobile

* Hero image becomes smaller
* Hero content and image rearrange vertically
* Category cards stack vertically
* Newsletter form changes into a column layout
* Product cards become full width
